## Hi, I'm Pranaya Khadka 👋

I'm a Machine Learning Developer and enthusiast from Kathmandu, Nepal. I work primarily with Python and deep learning frameworks, and I'm deeply focused on building models that solve real problems — not just notebooks that collect dust. I'm self-taught, open-source curious, and driven by a genuine obsession with how machines can learn, perceive, and reason.

Beyond ML, I think a lot about data, systems, and the math underneath it all. I believe in building things from scratch to truly understand them.
